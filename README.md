## What is it
Codyx is our novel program which is designed to be a secure and easily scalable solution that implements the robust security of blockchain into a simple ID proof. Our end goal is to ensure that everyone has the identification needed to avail opportunities.

## Why we built it
The reason we created Codyx was that we wanted to rectify the issue of lack of identification which affects many people. We don't want a simple thing like an ID to gatekeep people away from the biggest necessities. We felt strongly about this and thus decided to take matter into our own hands.
